# StudyFlow – High Fidelity UI Design

## CodeAlpha UI/UX Design Internship

### Task 2: High-Fidelity UI Design

### Project Overview

StudyFlow is a student productivity and study management mobile application designed to help students organize their learning activities efficiently. The application provides features for schedule management, assignment tracking, progress monitoring, and profile management through a clean and user-friendly interface.

### Objective

The objective of this project is to convert low-fidelity wireframes into a modern high-fidelity UI design by applying UI/UX principles, color theory, typography, icons, and visual hierarchy.

### Screens Designed

1. Splash Screen
2. Login Screen
3. Dashboard
4. Study Schedule Screen
5. Assignment Tracker Screen
6. Progress Tracker Screen
7. Profile Screen

### Design Elements Used

* Modern Purple Color Palette
* Clean Typography
* Rounded Cards and Buttons
* User-Friendly Navigation
* Consistent Icon System
* Responsive Mobile Layout
* Minimal and Professional UI Design

### Tools Used

* Figma
* Material Icons
* UI/UX Design Principles

### UI/UX Principles Followed

* Consistency
* Simplicity
* Visual Hierarchy
* Accessibility
* User-Centered Design
* Easy Navigation
